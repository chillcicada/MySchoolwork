## 第一章：视运动

> 建议：完全了解这一章的名词，尤其 2-3 小节的，自己画天球图。

1. 星座：仅代表天上的区域划分，不代表其中的星星存在内在联系
2. 天体的周日视运动
  - 成因：地球自转
  - 规律：天顶、子午线与地平面之间关系固定不变，某星相对于它们以天为周期，沿着与天赤道面平行的路径由东向西变化
  - 观测者角度：天球自东向西转动，所处纬度决定可见的天球部分、所处经度决定看见的星空的时间
3. 天体的周年视运动
  - 成因：地球绕太阳公转
  - 规律：以恒星年为周期，太阳在天球上的位置自西向东移动回到原点形成黄道
  - 观测者角度：以年为周期，太阳每天相同时刻的高度位置连续变化；天体每天比前一日提前约 4 分钟到达相同位置，星空复原周期为一个恒星年。
  - **四季更替成因**：地球绕太阳公转时，自转轴的倾斜（天赤道与黄道面的夹角为 23.5°）改变了阳光照射在不同位置的地球表面的角度，阳光角度的变化和一天的不同导致了地球上的季节变化。
4. 地球自转轴的进动
  - 规律：周期约 25800 年。北天极以北黄极为中心画图9黄道固定而天赤道移动） => 春（秋）分点沿天赤道由东向西巡回一圈
  - 现象：每年的春（秋）分点提前一点来临；赤经和赤纬每 100 年变化约 1.4°
5. 月球运动与月相
  - 规律：月球自转周期 = 月球公转周期 => 月球总是同一面向地球
  - 现象：不同光照导致月相：新（满）月至新（满）月时间间隔月 29.53 天（会合周期）
6. 日食：地球通过月球的阴影
  - 日全食、日环食：月球轨道非正圆导致月球视运动大小不同，全过程约 90 分钟
  - 月球影子小 => 发生概率小；可精确观测
7. 月食：月球通过地球的阴影
  - 与上类似。但全过程约 100 分钟，且发生概率较大。

## 运动

> 建议：了解图像

1. 日心说
  - 行星逆行原因：小轨道行星（地球）比大轨道行星绕日公转更快 => 前者“追上并超越”后者
  - 亮度变化原因：行星到地球的距离变化
2. 开普勒行星运动三规律
  - 行星以椭圆轨道环绕太阳运行，太阳（近似）位于椭圆的一个焦点上
  - 行星和太阳的连线在相等的时间内扫过相等的面积 => 行星越接近太阳则运行速度越快
  - $P^2\proptoA^3$；太阳系：$P_{year}^2=A_{AU}^3$
3. 伽利略的主要天文发现
  - 首次使用望远镜观测星空并彻底否定地心说，
  - 发现金星越亮则看起来越远，彻底证明日心说；
  - 月球表面不光滑；
  - 太阳黑子、太阳自转；
  - 银河中存在大量恒星；
  - 木星的4颗卫星（伽利略卫星）
4. 牛顿的引力理论
  - 牛顿三大运动定律：惯性定律、$F=ma$、作用力 = 反作用力
  - 万有引力定律：$F_g=\cfrac{Gm_1m_2}{r^2}$
    - 轨道： 由相关物体的引力决定。行星正圆轨道对应 $v_{circ}=\sqrt{GM/r}$，$M$ 为围绕的天体的质量、$r$ 为两者间距。
    - 逃逸速度：$v_{esc}=\cfrac{2GM}{r}=\sqrt{2}v_{circ}$
      - $v<v_{esc}$: 束缚椭圆轨道
      - $v=v_{esc}$: 非束缚抛物线轨道
      - $v>v_{esc}$: 非束缚双曲轨道
  - 潮汐：源自其他天体对其不同部分的引力作用差异
    - 潮汐力 = 地球每点受该天体引力和平均引力之差 = $f_{潮汐}\propto\cfrac{M}{r^3}$
      => 潮汐周期：与地球自转同向，约 25 小时（约 2 次涨落/天），主要源自月球绕地球公转，太阳起较小作用
    - 月球对地球的潮汐隆起导致地球自转变慢、月球轨道速度增加因而远离地球
    - 地球对月球的潮汐锁定导致月球同步自转
      两体近似：太阳 $\gg$ 行星 => 仅考虑太阳引力作用下的行星轨道的运算；行星 $\gg$ 卫星 => 仅考虑行星引力作用下的卫星轨道的运算
      - 偏离：引力摄动，其他质量对该近似的偏离
        - 可能代表存在未发现的行星，例：天王星 => 海王星的发现
        - 万有引力定律需要修正，例：水星 => 广义相对论
5. 狭义相对论
  - 基本假设：
    - 狭义相对论原理：一切物理定律在所有惯性参考系中形式相同，不存在特殊的绝对惯性系
    - 光速不变原理：在所有惯性系中，光在真空中的速度都是$c$，与光源和观察者的运动无关
  - 时空观：长度、时间（和质量）的测量都相对于观测者的运动，时间和空间之间协同构成四维时空
  - 现象：相对论速度（大于 ~0.1$c$）下匀速直线运动的物体：静止观测者发现该物体
  - 结论：$\gamma=\cfrac{1}{\sqrt{1-\cfrac{v^2}{c^2}}}\ge1$
    - 长度收缩：$L_{动}=L_{静}/\gamma$
    - 时间延缓：$\Delta t_{动}=\gamma\Delta t_{静}$
    - 质量（惯性）增加：$m_{动}=\gamma m_{静}$
  - 质量增加 => 加速度不再维持 => 光速是物体可获得的最快速度
  - 证据：$\mu$介子之谜
6. 广义相对论： 质量 $\Leftrightarrow$ 时空弯曲
  - 基本假设：
    - （弱）等效原理：惯性力与引力的力学效应在局域内等效 $\Leftrightarrow$ 等效的匀加速度（惯性力） = 引力加速度
    - 广义相对论原理：在描述物理定律时，所有参考系是等价的，不存在惯性系的特殊地位
  - 结论
    - 光线在强引力场中（在加速度下）弯曲——光沿测地线走 => 加速度导致时空弯曲
    - 引力使钟慢尺缩 => 质量导致时空弯曲
  - 证据：非欧几何、水星轨道进动、引力透镜、靠近大质量天体的时钟变慢、引力红移（2020 Nobel）、引力波
