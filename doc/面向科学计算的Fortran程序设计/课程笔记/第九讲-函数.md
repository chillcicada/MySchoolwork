### 子程序

子程序 (SUBROUTINE)：写程序时，可以把某一段常被使用、具有特定功能的程序代码封装成子程序。后续只需要通过该子程序的 CALL 命令，就能使用这段代码的功能，如：

```Fortran
program array
  implicit none
  call message("first")
  call message("second")
end program array

subroutine message(hint)
  implicit none
  character(len=*), intent(in) :: hint
  write(*,*) "get message: ", hint
end subroutine message
```
